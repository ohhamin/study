# 클린코드

클린코드란 가독성이 높은 코드를 말한다.

가독성을 높이기 위해선

- 네이밍이 잘 되어야 함

- 오류가 없어야 함

- 중복이 없어야 함

- 의존성을 최대한 줄여야 함

- 클래서 혹은 메소드가 한 가지 일만 처리해야 함

### 리팩토링

프로그램의 외부 동작은 그대로 둔 채, 내부의 코드를 정리하면서 개선하는 것을 말함

리팩토링이 필요한 코드는

- 중복 코드

- 긴 메소드

- 거대한 클래스

- Switch 문

- 절차지향으로 구현한 코드

리팩토링의 목적은 소프트웨어를 더 이해하기 쉽고 수정하기 쉽게 만드는 것이다.
